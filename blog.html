local storage vs session storage

Session storage is similar to Local storage. The difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends. Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab.

block scope vs global scope

Function Scope: When a variable is declared inside a function, it is only accessible within that function and cannot be used outside that function.
Block Scope: variable when declared inside the if or switch conditions or inside for or while loops, are accessible within that particular condition or loop.

What is an event loop in JS?

Event loop is an endless loop, which waits for tasks, executes them and then sleeps until it receives more tasks. The event loop executes tasks from the event queue only when the call stack is empty i.e. there is no ongoing task. The event loop allows us to use callbacks and promises.

ways to get undefined?

The short answer is that JavaScript interpreter returns undefined when accessing a variable or object property that is not yet initialized. There are many scenarios that can causes undefined. such as

    i. Uninitialized variable
    ii. Accessing a non-existing property
    iii. undefined function parameters
    iv. returning undefined value from a function
    v. avoiding operators. etc
